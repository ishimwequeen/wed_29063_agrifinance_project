# AgriFinance Subsidy Management System

##  Project Overview
A comprehensive PL/SQL Oracle database solution for managing agricultural subsidies, automating application processes, and providing real-time tracking for farmers and government officials.

##  Problem Statement
Agricultural subsidy distribution in Rwanda faces significant challenges:
- Manual paper-based application processes causing 30+ day delays
- Lack of transparency in approval and disbursement
- No real-time tracking for farmers
- Inefficient data management for policy decisions
- Limited audit trails for accountability

---

##  Solution
**AgriFinance Subsidy Management System** - A production-ready Oracle database solution that:
- Automates the entire subsidy lifecycle
- Provides real-time application tracking via SMS
- Ensures complete audit compliance
- Enables data-driven policy decisions
- Reduces processing time from 30 days to 3 days

---

##  Database Architecture

### Core Tables
FARMERS:  Registered farmer information 
SUBSIDY_TYPES: Different subsidy categories 
APPLICATIONS: Subsidy applications 
LANDS: Farmer land registrations 
CUSTOMER_MESSAGES: SMS/notification system 
SUPPLIERS: Approved suppliers 
BANKS: Payment bank partners 
PAYMENTS:SHOWS SUBSIDIES GIVEN OUT
APPLICATIONS: WHO APPLIED FOR A SUBSIDY
FARMER_ALERT_LOG: 
PUBLIC_HOLIDAY: SHOWS PUBLIC HOLIDAYS
REPEAT_ERROR_ALERTS:SHOWS ERRORS REATED MORE THAN 3 TIMES
